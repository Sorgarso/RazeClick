# RazeClick - Your Best Choice.

Windows auto clicker with multi-location support and precise CPS control.

<img width="800" height="215" alt="image" src="https://github.com/user-attachments/assets/2a9d8db3-b104-4674-840b-867e87149229" />

## Features

- Toggle / Hold activation modes
- Left or right mouse button
- Precise CPS control (up to 500 clicks/sec)
- Multi-location clicking (user-defined coordinates)
- Always on top
- Customizable hotkey (F1–F12)

## Usage

1. Select mouse button and activation hotkey
2. Choose mode: Toggle or Hold
3. Set click count and period
4. Add click locations (optional)
5. Press hotkey to start/stop
